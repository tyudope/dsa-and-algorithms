# Data Structures & Algorithms

Implementations of common data structures and algorithms, with notes and solved problems.

## Structure

```
dsa/
├── roadmap.md             # topic sequence + progress tracker
├── data-structures/       # implementations + notes
├── algorithms/
├── patterns/
└── leetcode-problems/     # solutions with write-ups
```

- **data-structures/** — arrays, linked lists, stacks, queues, hash tables, trees, heaps, graphs. Each implemented from scratch with accompanying notes.
- **algorithms/** — sorting, searching, graph traversal, dynamic programming.
- **patterns/** — common problem-solving patterns: two pointers, sliding window, binary search, backtracking.
- **leetcode-problems/** — solved problems, each with a short write-up of the approach and complexity.

## Language

Python.
